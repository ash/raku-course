Here is a possible solution to the task.

## Code

```raku
say [+] 10, 20, 30, 40;
```

🦋 You can find the source code in the file [sum-list.raku](https://github.com/ash/raku-course/blob/master/exercises/paradigms/reduction/sum-list.raku).

## Output

```
100
```

## Comments

1. `[+]` inserts `+` between every element of the list.

1. The sum `10 + 20 + 30 + 40` is `100`.
