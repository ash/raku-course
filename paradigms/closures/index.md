title: Lambdas and closures

{% include menu.html %}

A _lambda_ is a small, nameless function written inline — exactly the pointy blocks and Whatever expressions you have been passing to `map` and friends. A _closure_ is a lambda (or any subroutine) that remembers variables from the place where it was created.

Closures are what make returned subroutines so useful: they carry a little piece of private state with them. This section looks at both ideas.
