## Problem

Given the list of words `<apple pear plum>`, start **one promise per word** that upper-cases it, so the work runs concurrently. Await all of the promises together and print the resulting list of upper-cased words.

## Example

The program prints:

```
(APPLE PEAR PLUM)
```
