## Problem

Using the sequence operator with a closure, build the Tribonacci sequence — like Fibonacci, but each number is the sum of the previous **three**, starting `1, 1, 1`. Print its first eight numbers.

## Example

The program prints:

```
(1 1 1 3 5 9 17 31)
```
