## Problem

A palindrome reads the same forwards and backwards. For each of the words
`level`, `hello`, and `racecar`, print the word followed by `yes` or `no`
according to whether it is a palindrome.

## Example

The program prints:

```
level: yes
hello: no
racecar: yes
```
