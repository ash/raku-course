## Problem

Encrypt the upper-case word `HELLO` with a Caesar cipher that shifts each letter
three places forward in the alphabet, wrapping `X`, `Y`, `Z` around to `A`, `B`,
`C`. Print the encrypted word.

## Example

The program prints:

```
KHOOR
```
