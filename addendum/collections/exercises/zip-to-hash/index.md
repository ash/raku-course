## Problem

You have two parallel lists: names `<Anna Bob Cara>` and scores `90, 85, 95`.
Combine them into a hash mapping each name to its score, and print each entry
as `name: score`, ordered by name.

## Example

The program prints:

```
Anna: 90
Bob: 85
Cara: 95
```
