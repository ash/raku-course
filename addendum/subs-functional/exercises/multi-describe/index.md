## Problem

Using multiple dispatch, write a `describe` subroutine that reacts differently to
an integer, a string, and a list. Call it with `42`, `'hi'`, and `[1, 2, 3]`.

## Example

The program prints:

```
integer 42
string hi
list of 3
```
