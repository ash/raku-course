## Problem

Turn each of the scores `95`, `82`, `71`, and `58` into a letter grade: `A` for
90 and above, `B` for 80–89, `C` for 70–79, `D` for 60–69, and `F` below that.
Print each score with its grade.

## Example

The program prints:

```
95 -> A
82 -> B
71 -> C
58 -> F
```
