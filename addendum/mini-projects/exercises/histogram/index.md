## Problem

Given sales figures `apples => 5, pears => 3, plums => 8`, print a simple bar
chart: each label, then a tab, then a bar of `#` characters as long as its value,
ordered by label.

## Example

The program prints:

```
apples	#####
pears	###
plums	########
```
