## Problem

Group the words `<listen silent enlist cat act dog>` so that anagrams (words made
of the same letters) land together. Print each group on its own line, words in
alphabetical order, groups ordered by their shared letters.

## Example

The program prints:

```
act cat
dog
enlist listen silent
```
