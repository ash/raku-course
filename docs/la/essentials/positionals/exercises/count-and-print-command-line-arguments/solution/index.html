Solutio: Numerare et print argumentorum lineae mandati

In hoc programmate, for ansa bona electio est.

Codex

for ^@*ARGS -> $n {
    say "{$n + 1}. @*ARGS[$n]";
}

Disputatio

Hoc programma paucae artes utiles utitur. Primo, for ansa pergit per extensionem constructam cum ^. Ita, extensio incipit ab 0 et pergit usque ad (sed non includens) valorem @*ARGS. In hoc contextu, redit longitudines array.

Cum primum elementum array indicem 0 habeat, et opus sit ut lineas a 1 numeremus, haec simplex translatio computatur intra codicem clausulam in chorda: "{$n + 1} ...".
