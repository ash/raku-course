Solution: Odd or even

The body of the function is quite small, which is a valid reason to omit both the return routine and the semicolon at the end of the line.

Code

Here is the solution:

sub f($n) {
    $n %% 2 ?? 'Even' !! 'Odd'
}

say f(10);
say f(5);
say f(0);
say f(3);
