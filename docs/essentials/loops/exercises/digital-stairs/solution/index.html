Solution: Digital stairs

The program uses two loops and two ranges to loop over them.

Code

Here is one of the possible solutions:

my $size = prompt 'Enter the size: ';

for 1..$size -> $n {
    .print for 1..$n;
    print "\n";
}

Comment

Note how the current digit is printed:

.print for 1..$n;

As with say, the print routine can be called as a method. In this case, it is called on the topic variable $_.
