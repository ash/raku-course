Built-in functions for printing

To print data to the console (or, more strictly, to the STDOUT stream), we used the built-in routine say a lot. There are a few more built-in functions that can be used for printing. We’ve briefly seen most of them in the examples of ‘Hello, World!’ already. In this section, more details are given.
