Ternary operator

Ternary operator in Raku is a two-part construction ?? â€¦ !!. A Boolean test is followed by two expressions, one of which is executed depending on the result of the test.

my $shines = 'Sun';
my $day-time = $shines eq 'Sun' ?? 'day' !! 'night';
say $day-time;

With the current value of the $shines variable, this program prints day. If you change it to Moon, the result will be night.
