else blocks

An if block can be combined with an else block, which is run when the condition is not satisfied.

my $t = 36.6;
if 35.5 < $t < 37.5 {
    say 'You are fine.';
}
else {
    say 'Call a doctor, maybe?';
}

The else block cannot be used by its own (there is a better alternative called unless, which will be introduced just in a minute).
