Running programs

In one of the previous sections, we created a simple program, but did not run it yet. There are four main ways of running the program: using an installed command-line tool, using a REPL shell, running a program directly from an IDE or using an online service such as glot.io.

There are two five in this section. You can browse all of them or examine only those that you find suitable for your needs. After the sections, there is an assignment.

Topics in this section

  • Running from command line
  • Running from REPL
  • Running from IDE
  • Using docker
  • Using online services
