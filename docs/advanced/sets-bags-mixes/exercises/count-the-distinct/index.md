## Problem

You are given a list of numbers, some of which repeat:

```raku
my @data = 3, 1, 4, 1, 5, 9, 2, 6, 5, 3;
```

Print how many _distinct_ values it contains. Use a set.

## Example

The program prints:

```
7
```
