## Problem

You are given two arrays:

```raku
my @first = 1, 2;
my @second = 3, 4, 5;
```

Combine them into a single _flat_ array (not a nested one). Print the array and then the number of its elements.

## Example

The program prints:

```
[1 2 3 4 5]
5
```
