## Problem

Write a Pod block with a `=head1` heading `Greeting` and a paragraph `Hello, world!` in which the word `world` is marked bold with `B<…>`. Render it with `raku --doc`.

> Run the file with `raku --doc` to see the rendered text.

## Example

Rendered with `raku --doc`, it prints:

```
Greeting

Hello, world!
```
