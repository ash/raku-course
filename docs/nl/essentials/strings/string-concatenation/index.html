String concatenatie

Als je twee strings hebt, kun je ze samenvoegen en een nieuwe, langere string krijgen. Deze actie wordt stringconcatenatie genoemd. In Raku concateneer je strings metâ€”gok eens?â€”de concatenatie-operator. Deze operator is een tilde: ~. Om twee strings te concateneren, plaats je ~ ertussen:
