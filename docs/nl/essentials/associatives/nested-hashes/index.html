Geneste hashes

Het maken van geneste hashes is vergelijkbaar met het maken van geneste arrays. Gebruik accolades om de binnenste hashes te omarmen.

my %people =
    John => {
        age => 20,
        city => 'Madrid'
    },
    Alla => {
        age => 21,
        city => 'Tokyo'
    };

Merk op dat een afsluitende komma een geldig syntactisch element is:

my %people =
    John => {
        age => 20,
        city => 'Madrid',
    },
    Alla => {
        age => 21,
        city => 'Tokyo',
    };
