Oplossing: Faculteit

Een faculteit van N is een product van gehele getallen van 1 tot en met N. Bijvoorbeeld, de faculteit van 4 is 1 * 2 * 3 * 4 = 24.

Code

my $n = 8;

my $f = 1;
$f *= $_ for 2..$n;

say $f;
