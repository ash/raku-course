note

De note routine print zijn argumenten naar de standaard foutstroom. Verder is het vergelijkbaar met say.

  1. Roept de gist methode aan op zijn argumenten.
  2. Voegt een nieuwe regelkarakter toe.
  3. Converteert het resultaat naar UTF-8.
  4. Stuurt het naar de STDERR stroom.
my $x = 42;
note "Huidige waarde van \$x is $x";

Bevestig dat dit programma het bericht niet naar STDOUT stuurt:

$ raku t.raku > /dev/null
Huidige waarde van $x is 42
