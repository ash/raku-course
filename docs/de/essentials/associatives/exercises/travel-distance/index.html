Problem

my @itinerary = < Berlin Prag Wien Zürich Paris >;

Beispiel

Für die gegebene Liste von Städten sieht die Ausgabe des Programms wie folgt aus:

Prag — Wien: 312 km
Wien — Zürich: 784 km
Zürich — Paris: 557 km
Gesamte Reise: 2007 km
