title: Anchors

{% include menu.html %}

So far a pattern could match anywhere inside a string. An _anchor_ does not match a character at all — instead it asserts a **position**: the start of the string, the end, or the boundary of a word.

Anchors are what let you say “the string must start with this”, “the whole string must be exactly this”, or “this must be a complete word, not just part of one”.
