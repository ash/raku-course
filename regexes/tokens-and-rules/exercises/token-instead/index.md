## Problem

Show the difference backtracking makes. Declare the **same** pattern — one or more digits followed by a literal `5` — twice: once as a `regex` and once as a `token`. Match each against `'12345'` and print a plain `True` or `False` for each.

## Example

The program prints:

```
True
False
```
