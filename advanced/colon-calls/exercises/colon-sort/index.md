## Problem

Using the colon call form, sort the words `pear`, `fig`, `apple`, `kiwi` by their **length**, from shortest to longest. Pass the sort key as a Whatever expression rather than a block. Print the result.

## Example

The program prints:

```
(fig pear kiwi apple)
```
